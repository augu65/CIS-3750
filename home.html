Welcome

Welcome to the administration portal for the WDGPH tick identification phone app. Through here you can reskin the app, manage administration accounts, change the questionnaire, and change the educational info that will be displayed on the home page of the app.

The Reskin page allows you to update the design of both the app and the webpage. This can be done by changing the logo, colours, font, and font size. Manage Accounts allows you to update, create, delete, and archive accounts of the web portal. Manage Questionnaire allows you to update, create, and delete questions that are shown to the user of the app. Educational Page allows you to update, create, and delete articles, pictures, and links that is shown to the user of the app in the educational section.

All changes that are made though this administration web portal effect the app called TickedOff, the WDGPH app for submitting tick images.
